Welcome to the JD Weather API

The JD Weather API is a REST Server which allows you to fetch weather data from multiple divverent available weather data services.

Registration

You must be a registered user of the JD Weather API, and be a registered user of each weather service you wish to use. Please click the "Register" button on the top-right corner of this page to register for a new account.

NOTE: The registration option is currently not available. Please contact support to acquire access.

Supported Services

Sending Requests

All requests are sent to the API using the following base URL:

http://api.weather.jdsoftwareinc.com/[YOUR_KEY]/

Multiple endpoints can be combined in a single request:

http://api.weather.jdsoftwareinc.com/[YOUR_KEY]/[ENDPOINT_ONE]/[ENDPOINT_TWO]/

Endpoints

  • Services
  • Support
  • Conditions
  • Alerts
  • ForecastSummary
  • ForecastHourly
  • ForecastDaily
  • Maps

Endpoint: Services

Returns an array of all available services and their basic info.

http://api.weather.jdsoftwareinc.com/[YOUR_KEY]/Services/

Attribute Type Description
caption string User-friendly name of service
name string Unique internal name of service
uid string Unique internal identifier of service
url_main string Main URL of Weather Service
url_api string API Documentation URL of Weather Service
url_login string Login URL of Weather Service
url_register string Register URL of Weather Service
